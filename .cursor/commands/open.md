# Open（ファイルを開く）

目的:
- 指定したファイルを、現在のエディタで開ける形で提示する。

使い方:
- `/open <path>`
- `/open <path>:<line>`
- 複数指定する場合は改行で列挙する。

実行手順:
1. 入力されたパスを解決する（ワークスペース相対パスも許可）。
2. まず Markdown のローカルリンクを返す。
3. ユーザーが「開けない」と返した場合のみ、対象ファイルに最小の一時変更を入れて変更通知カードを出す。
4. ユーザーが開けたことを確認したら、一時変更を削除して元に戻す。

制約:
- 既存内容を壊さない。
- 一時変更は必要時のみ行う。
- 文字コードは UTF-8（BOM なし）、改行は LF を維持する。

出力形式:
- 先頭に開くためのリンク一覧を出す。
- 一時変更を行った場合だけ、変更したファイルを最後に列挙する。
