# ✔️ 絶対死守！回帰チェック最強手順書 ✔️

アプデのたびにアプリがぶっ壊れる恐怖とはおさらばだ！
このチェックリストは、アプリの「命綱」！コードをいじったら、絶対に絶対にこれをやって「無事」を確かめるんだ！🔥

## 1. 🎯 この儀式の目的
- アプリの心臓部（起動、取込、サムネ、検索）が元気に動いてるか、「秒」で確認する！⏱️
- アホなバグをユーザー（自分）の顔面にぶつける前に、この壁で100%撃ち落とす！🔫

## 2. 🧰 準備するもの
- `Debug|x64` でのビルドがバシッと通っていること！
- 軽くて可愛い「テスト用動画フォルダ」（5〜20個くらい）を作っておく！📁
- 実験台にされる「テスト用DB」（新規でも適当なやつでもOK）を1つ用意！

---

## 3. 🚨 クリティカルチェック（死守帯）

### 3.1 🚪 起動・終了（息してる！？）
1. アプリをバーンと起動！
2. メイン画面がドーンと表示される！
3. 必要ならテストDBを開く。
4. アプリをスッと終了する。

**🏆 期待する結果**:
- 起動時に意味不明なエラーで爆発しない！💥
- 終了時も静かに眠りにつく！
- 次回起動した時に「ウィンドウの場所とサイズ」「最後に開いたDB」をしっかり覚えてる天才っぷりを発揮！🧠

### 3.2 🗂️ DBオープン・一覧表示（目は見えてる！？）
1. DBを開く！
2. タブ（Small/Big/Grid/List/5x2）をDJのように次々と切り替える！🎧
3. 最初の動画をポチッと選んで、横の詳細ペインを見る！

**🏆 期待する結果**:
- 動画の一覧がズラッと並ぶ！
- タブを切り替えても固まらない！サクサク！✨
- 詳細ペインにタイトル・パス・タグがバッチリ表示されてる！

### 3.3 📥 監視フォルダ取込（耳は聞こえてる！？）
1. さっき用意した監視フォルダを1個登録！（watch=ONにするの忘れるな！）👀
2. アプリを起動したままの状態で、そのフォルダに動画をポイッと放り込む！
3. 動画一覧と「〇〇件」の表示を凝視する！

**🏆 期待する結果**:
- 追加した動画がDBにシュバッと登録される！
- 画面の一覧にもサッと現れる！神速！⚡
- ファイルを入れた瞬間にアプリがパニックになってクラッシュしたりしない！

### 3.4 🖼️ サムネイル生成（絵は描ける！？）
1. 一覧から適当な動画を選ぶ。
2. 気合を入れて「等間隔サムネイル作成」ボタンをターンッ！🔘
3. 進捗のバーや通知が動くのを見守る。

**🏆 期待する結果**:
- ピカピカのサムネ画像が出来上がる！✨
- （動画がちゃんとあるのに）ずーっとエラー画像のまま拗ねたりしない！
- サムネを作ってる最中でも、画面全体が重すぎてマウスがカクカクにならない！💪

### 3.5 🔍 検索・履歴（記憶力はどう！？）
1. 検索窓に思いついた言葉（タグとか）を入れてみる。
2. 動画の件数が減って絞り込まれるのを見る！
3. `Enter` キーをッターン！と叩いて履歴に記憶させる！
4. 検索窓の▼を押して、さっきの履歴を選ぶ！
5. `Delete` キーを押して、その履歴を消し去る！🧹

**🏆 期待する結果**:
- 入力した言葉でちゃんと検索結果が絞り込まれる！
- 履歴が「登録」「選択」「削除」の全部できる！完璧！
- 日本語を打ってる最中（変換中）に勝手に検索が発動してイライラさせない！😡

### 3.6 🏷️ タグ操作（整理整頓できる！？）
1. タグを追加！編集！そして削除！やりたい放題やる！🗑️
2. タグをクリックして単体検索！`Ctrl` を押しながらクリックして複数条件での検索！

**🏆 期待する結果**:
- いじったタグが、画面にもDBにも即座に反映される！
- タグをクリックしたら、ちゃんとそのタグで検索された状態になる！賢い！🧠

### 3.7 ▶️ 再生・ブックマーク（一番楽しい時間！？）
1. 動画をダブルクリックして普通に再生！🎬
2. カッコいいシーンで「ブックマーク追加」！！🔖
3. 作ったブックマークからもう一度再生！
4. 飽きたらブックマークを消去！

**🏆 期待する結果**:
- 指定したお気に入りのプレイヤーがバシッと立ち上がる！🚀
- スコア（☆）や再生回数、最終再生日がこっそり更新されてる！
- ブックマーク一覧に、ちゃんと追加と削除が反映される！

### 3.8 🔄 ファイル操作（引越しできる！？）
1. アプリ内でファイル名をリネーム！✏️
2. 気分が乗ったら、コピー、移動、削除もやってみる！
3. 動画一覧とサムネイルが、ちゃんと新しいファイル構成についてきてるか確認！

**🏆 期待する結果**:
- DBの `movie_name` や `movie_path` が新しい情報に書き換わってる！
- ファイルを動かしても画面がバグったり破綻したりしない！
- フォルダ監視（Watcher）が動いててファイルをつまみ食いしてても、致命的なエラーで死なない！🛡️
