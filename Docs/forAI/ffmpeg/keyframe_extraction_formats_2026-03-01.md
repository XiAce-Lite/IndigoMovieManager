# 🎯 動画形式別！最高効率のキーフレーム（Iフレーム）抽出テクニックまとめ

動画からサムネをぶっこ抜くなら、キーフレーム（Iフレーム）を狙い撃ちするのが最強！🔥
でも、動画コンテナやコーデックによってFFmpegの挙動が変わるっていう厄介な罠があるんだ。
そんなコンテナ・コーデック別の「キーフレーム抽出の正解」をまとめたよ！✨

## 🚀 共通の基本戦略：「ダブル `-ss`」と「Iフレーム間引き」
まずは基本のおさらい！どんな形式でも、基本的には以下のコマンドがベースになるよ👍

### ① 爆速タイムスタンプ抽出（ダブル `-ss`）
1枚だけ抜きたいなら、キーフレームを利用したワープで目的地付近へ飛んでから微調整！
```bash
ffmpeg -ss 55.0 -i input.mp4 -ss 0.2 -frames:v 1 output.jpg
```

### ② サムネ量産特化（Iフレームのみ抽出）
シークバー用のサムネを作るなら、重いデコード処理をスキップしてIフレームだけを抽出！
```bash
ffmpeg -i input.mp4 -vf "select='eq(pict_type,I)'" -vsync vfr -q:v 2 thumb_%04d.jpg
```
※Iフレームが多い場合は `select='eq(pict_type,I)*not(mod(n,10))'` みたいにモジュロ演算で間引いてね🥰

---

## 🎬 コンテナ・コーデックごとの罠と最適解

動画フォーマットごとに内部構造の仕様が違うから、上手く抜けない時は以下を参考にしてね！

### 📦 MP4 / MOV (H.264, HEVC)
* **特徴:** 最も素直で扱いやすい優等生！
* **注意点:** 可変フレームレート（VFR）になっていることが多いから、`-vsync vfr` を付けないと動画長と抽出枚数がズレてカオスになるよ😇
* **最強手順:** 基本戦略のコマンドがそのまま最高のパフォーマンスを出すよ！

### 📦 MKV (Matroska)
* **特徴:** トラックが何重にもなってるアーカイブ界の王様👑
* **注意点:** 複数の映像ストリームや字幕が入っていることがある！
* **最強手順:** `-map 0:v:0` で「最初のビデオストリームだけ」を明示的に指定して無用なトラブルを防ごう！
```bash
ffmpeg -i input.mkv -map 0:v:0 -vf "select='eq(pict_type,I)'" -vsync vfr thumb_%04d.jpg
```

### 📦 AVI / FLV (H.263, レガシーH.264など)
* **特徴:** ザ・レガシー！昔の設計だからメタデータが適当なことが多い👴
* **注意点:** 古いAVIだとインデックス（フレームの位置情報）が破損してて、最初のワープ（`-ss`）が効かない・ズレることがある！
* **最強手順:** `ffprobe` であらかじめキーフレームの位置を抜いておくか、`-ss`をファイルの**後**に置いて力技でデコードする（時間はかかるけど確実！）のが安全👍

### 📦 TS (MPEG-TS)
* **特徴:** 放送用！途切れやエラーに強いかわりに、インデックス情報を持たない仕様になっている💥
* **注意点:** ファイルの先頭に「全体の目次」がないから、入力前の `-ss`（Fast Seek）が機能せず、シークが激遅になるよ😱
* **最強手順:** TSファイルからキーフレームを抜くなら、一旦 `-c copy` でインデックスのあるMP4に「リマルチプレックス（再パッケージ化）」してから処理するのが結果的に最速！🔥
```bash
# 1. 爆速で目次付きのMP4に変換（再圧縮なし）
ffmpeg -i input.ts -c copy temp.mp4
# 2. そのMP4からIフレームを抜き出す
ffmpeg -i temp.mp4 -vf "select='eq(pict_type,I)'" -vsync vfr thumb_%04d.jpg
```

### 🎥 ProRes RAW / 特殊な中間コーデック
* **特徴:** 超高品質なプロ向けフォーマット！ファイルサイズが尋常じゃない🐘
* **注意点:** そもそも「フレーム間圧縮（I/P/Bフレームの概念）」をしておらず、「全フレームがIフレーム（All-Intra）」なのが特徴！
* **最強手順:** 全域Iフレームだから `pict_type,I` のフィルタは意味がないよ！普通にFPSフィルタ(`fps=1`とか)で時間ベースで間引くのが正解！✨
```bash
ffmpeg -hwaccel auto -i input.mov -vf "fps=1,scale=320:-1" -q:v 2 thumb_%04d.jpg
```

これさえ分かっていれば、どんな変態的な動画ファイルが来ても爆速でサムネをぶっこ抜けるね！最高じゃん！🚀
