# 🎨 アイコン作成バッチ（DropIconMaker）使いかたガイド

このガイドでは、`C:\SNLApp\icons\sjis_DropIconMaker_ImageMagik.bat` を使って、画像から Windows 用アイコン（.ico）を爆速で作成する方法を解説するぜ！🚀

## 1. 準備する物
- 変換したい画像ファイル（`.png` や `.jpg` など）
  - ※ 512x512 以上の正方形画像がオススメだぜ！✨
- **ImageMagick** (画像変換エンジン)
  - [Official Website: imagemagick.org](https://imagemagick.org)

## 2. 使い方（ドラッグ＆ドロップ）
1. **フォルダを開く**
   - `C:\SNLApp\icons\` を開いて、バッチファイルを確認してね。
2. **画像をバッチに落とす**
   - 変換したい画像ファイルをマウスで掴んで、`sjis_DropIconMaker_ImageMagik.bat` の上に直接 **ドラッグ＆ドロップ** しよう！
3. **アイコンの完成！**
   - 黒い画面（コマンドプロンプト）がシュパッと一瞬出て消えたら成功！
   - 元画像と同じフォルダに、`.ico` 拡張子のファイルが出来上がっているはずだぜ！どや！😎🔥

## 3. ワンポイントアドバイス 💡
- **同一フォルダが便利**: 元画像をバッチと同じ `C:\SNLApp\icons\` にコピーしてからドロップすると、結果が目の前に出るから爆速で確認できるよ。
- **マルチアイコン対応**: このバッチは ImageMagick を使って、256px から 16px までの複数のサイズを一つの `.ico` に詰め込んでくれる優れものだ！
  - これがあれば WPF のアプリアイコンも、WiX インストーラーのロゴもバッチリだね！🥰💖🚀
