## ヘッダー破損

### 劣化なしで修復する（推奨）
```bash
ffmpeg -i input.mp4 -c copy -map 0 -movflags +faststart output.mp4
```
-map 0: 全てのストリーム（映像・音声・字幕）を維持します。

+faststart: 目次を先頭に移動させ、アプリが即座にサムネイル用のフレームを見つけられるようにします。

### それでもサムネイルが作れない場合（再エンコード）
動画のヘッダーだけでなく、中のデータ配列（GOP）に問題がある場合は、キーフレームを強制的に挿入して再構成します。

```bash
ffmpeg -i input.mp4 -c:v libx264 -pix_fmt yuv420p -g 30 -c:a copy output.mp4
```
-pix_fmt yuv420p: 最も互換性の高い形式に変換します。

-g 30: 30フレームごとに必ずキーフレームを入れることで、アプリがサムネイルを切り出しやすくします。
