# 動画拡張子追加手順

## 目的
監視対象の動画拡張子を追加する時の手順をまとめる。  
現在の既定値は `CheckExt` で管理している。

## 変更箇所
1. `Properties/Settings.settings`
2. `App.config`

## 追加手順
1. `Properties/Settings.settings` の `CheckExt` に拡張子を追加する。
2. `App.config` の `CheckExt` に同じ拡張子を追加する。
3. 区切りはカンマ `,` を使う。
4. 書式は `*.webm` のように `*.` 付きでそろえる。
5. 既存順を大きく崩さず、末尾追加を基本にする。

## 例
```xml
<Value Profile="(Default)">*.avi,*.wmv,*.mp4,*.mkv,*.webm,*.ts,*.m2ts</Value>
```

```xml
<value>*.avi,*.wmv,*.mp4,*.mkv,*.webm,*.ts,*.m2ts</value>
```

## 注意
- `Settings.settings` だけ変更すると、既定値と実行時設定の見え方がずれることがある。
- `App.config` だけ変更しても、設定定義側とずれるので両方そろえる。
- 監視イベントと手動走査の両方が `CheckExt` を参照するため、追加後は再起動して反映を確認する。
