### 🤯 3万件で21GB!? メモリ爆食い事件とスマート対応の軌跡 (2026-03-25)
- [21GB爆食いの真相と解決策](Docs/Gemini/MemoryLeakIncident_2026-03-25.md) **【NEW】メモリ大ダイエット！✨**
- 3万件登録時にメモリが21GBまで跳ね上がる怪現象が発生！原因は「バグ（GC漏れ）」ではなく、「全件・多重の抱え込み（長寿命参照）」だと特定！
- 無駄な `.ToArray()` の配列コピーを「ストリーム処理（流しそうめん方式）」に変更し、用済みの巨大 `DataTable` を即座に `null` で手放す等、**最小の修正で劇的なピーク低下・GC回収効率アップ**を達成した激闘の記録！🔥
