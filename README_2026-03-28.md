# IndigoMovieManager README

最終更新日: 2026-04-04

## 1. これは何か

IndigoMovieManager は、WhiteBrowser 互換を重視しながら

- 動画一覧の管理
- 検索
- 監視フォルダからの自動取り込み
- サムネイル生成

を扱う WPF デスクトップアプリです。

この文書は、このコードベースに新しく入る人が最初に迷わず読み始めるための開発者向け README です。

まず前提として、この repo の目標と要件は次で固定します。

- [リポジトリ目標と要件_2026-04-04.md](C:/Users/na6ce/source/repos/IndigoMovieManager/リポジトリ目標と要件_2026-04-04.md)

## 2. 最初に知っておきたいこと

- この repo は WhiteBrowser 互換と `*.wb` 正本運用を守りながら進めます
- このブランチは UI を含む高速化と安定化の本線です
- 最優先はユーザー体感テンポです
- アプリ本体は `net8.0-windows` の WPF アプリです
- メイン DB は WhiteBrowser 互換の `*.wb` を使います
- サムネイル生成は UI と分離され、QueueDB 経由で非同期に動きます
- 失敗したサムネイル生成は FailureDb と RescueWorker で救済します
- release package には bundled rescue worker を同梱し、worker pin 情報を追える前提で運用します

## 4. 最初にやること

1. 開発環境を入れる
   - Visual Studio 2026
   - .NET 8 SDK
   - PowerShell 7.x
2. リポジトリを開く
3. 依存関係を復元する
4. `Debug | x64` でビルドする
5. 小さい DB と小さい動画フォルダで起動確認する
6. 変更前に `リポジトリ目標と要件_2026-04-04.md` と本線方針を再確認する

## 5. よく使うコマンド

```powershell
dotnet restore
dotnet build IndigoMovieManager.sln -c Debug -p:Platform=x64
dotnet run --project IndigoMovieManager.csproj -c Debug
```

## 6. どこに何があるか

- `Views/Main/`
  - メイン画面
- `ViewModels/`
  - 表示用データの整理
- `Watcher/`
  - 監視、差分反映、MainDB 登録
- `Thumbnail/`
  - アプリ本体側のサムネイル入口
- `src/IndigoMovieManager.Thumbnail.Engine`
  - サムネイル生成ロジック本体
- `src/IndigoMovieManager.Thumbnail.Queue`
  - QueueDB と FailureDb の本体
- `src/IndigoMovieManager.Thumbnail.RescueWorker`
  - 救済 worker
- `DB/`
  - メイン DB の基本アクセス
- `Tests/IndigoMovieManager.Tests`
  - テスト

## 9. 最初の動作確認

次の 4 つが通れば、最初の環境確認として十分です。

1. アプリが起動する
2. DB を開ける
3. 小さい監視フォルダを追加できる
4. 動画 1 本のサムネイルが作られる

## 10. 詰まった時の見方

- 一覧が出ない
  - メイン DB と `ViewModels/MainWindowViewModel.cs`
- 監視がおかしい
  - `Watcher/README.md` と `Watcher/MainWindow.Watcher.cs`
- サムネイルが出ない
  - QueueDB、FailureDb、`Thumbnail/MainWindow.ThumbnailCreation.cs`

## 11. 実装時の注意

- `*.wb` を変更する方向へ寄せない
- いきなり巨大ファイル全体を読まない
- まず触る領域を決める
- サムネイル周りでは `Factory + Interface + Args` の入口を守る
- UI スレッドへ重い処理を戻さない
- 通常動画の初動を、難読動画や救済都合で悪化させない
- 配布導線を触る時は bundled rescue worker の pin 情報を壊さない
