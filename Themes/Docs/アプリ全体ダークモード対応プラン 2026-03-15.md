# 🌙 アプリ全体ダークモード対応プラン 2026-03-15

## 現状
MainWindow.xaml は全色をDynamicResource化済み ✅
だけどそれ以外のUserControlや子画面にハードコード色が残ってて、ダークモードだと見えない部分がある 😅

## 対処が必要な箇所（優先度順）

### 🔴 即対応: Foreground="Black" を削除（6箇所）
ダーク背景だと**文字が見えなくなる**最大の問題！
削除すればOSテーマの文字色に追従する。

| ファイル | 行 | 要素 | 対処 |
|---------|---|------|------|
| TagControl.xaml | 40 | TagButton Label | `Foreground="Black"` 削除 |
| TagControl.xaml | 60 | removeTag Label | `Foreground="Black"` 削除 |
| Bookmark.xaml | 77 | ファイル名 Label | `Foreground="Black"` 削除 |
| ExtDetail.xaml | 93 | ファイル名 Label | `Foreground="Black"` 削除 |
| ExtDetail.xaml | 112 | 拡張子 Label | `Foreground="Black"` 削除 |
| ExtDetail.xaml | 132 | フォルダ Label | `Foreground="Black"` 削除 |

### 🔴 即対応: UnderlineBrush="Black" を削除（3箇所）
MaterialDesign の下線を黒固定すると、OS連動時に入力欄だけ浮いて見える。

| ファイル | 行 | 要素 | 対処 |
|---------|---|------|------|
| RenameFile.xaml | 70 | 変更後の名前 TextBox | `materialDesign:TextFieldAssist.UnderlineBrush="Black"` 削除 |
| RenameFile.xaml | 101 | 拡張子 TextBox | `materialDesign:TextFieldAssist.UnderlineBrush="Black"` 削除 |
| TagEdit.xaml | 40 | タグ編集 TextBox | `materialDesign:TextFieldAssist.UnderlineBrush="Black"` 削除 |

### 🟡 対応推奨: タグBorder色（1箇所）
`LightGreen` はダークモードでもそこそこ見えるが、テーマ連動にした方がベター。

| ファイル | 行 | 要素 | 対処 |
|---------|---|------|------|
| TagControl.xaml | 18-19 | Border Background/BorderBrush | `LightGreen` → DynamicResource `TagBorderBackground` に |

### 🟡 対応推奨: 下部タブの装飾色をテーマブラシへ統一
下部タブは固定グレーや半透明黒が残りやすい。`MaterialDesignCardBackground` / `MaterialDesignDivider` / `MaterialDesignBodyLight` へ寄せる。

| ファイル | 対処 |
|---------|------|
| Generic.xaml | 下部タブ共通サマリ枠を `MaterialDesignCardBackground` + `MaterialDesignDivider` へ |
| Generic.xaml | 下部タブルート用 `BottomTabRootUserControlStyle` で文字色を `MaterialDesignBody` へ統一 |
| Generic.xaml | `HighDensityBottomTabButtonStyle` / `DataGrid*Style` は `BasedOn` を付けて MDIX の既定スタイルを潰さない |
| DebugTabView.xaml | `GroupBox` / `Button` / `TextBox` のローカルスタイルを MDIX ベースへ変更し、素のWPF見た目を解消 |
| ThumbnailProgressTabView.xaml | 固定グレー文字・枠線・カード背景をテーマブラシへ置換 |
| ThumbnailErrorTabView.xaml | `要確認` サマリの直色背景を共通サマリスタイルへ統一 |

### ⚪ 除外: 状態表示色
意味のある色指定なのでそのまま。

| ファイル | 色 | 理由 |
|---------|---|------|
| RenameFile.xaml | `Red` | エラー表示用 |
| RenameFile.xaml / CommonSettings | `Gray` | 無効状態表示 |

## 実装手順
1. **Foreground="Black" を6箇所から削除**
2. **UnderlineBrush="Black" を3箇所から削除**
3. **TagControl の LightGreen を DynamicResource 化**
4. **下部タブの固定グレー/半透明色をテーマブラシへ置換**
5. OriginalColors.xaml / OsSyncColors.xaml にキー追加
6. ビルド確認
