## 進め方

1. 1 本ずつ表示可否を確認する
2. 表示不可なら、失敗点を次の 3 種で切る
   - callback 不足
   - DOM 前提差分
   - skin 固有 script 依存
3. 共通原因なら compat へ戻す
4. 個別原因なら skin 単位の補完メモを追記する

- 2026-04-12 時点で compat runtime には、`onCreateThum` 未実装向け最小サムネ生成 fallback と、`onUpdateThum` 未実装向け `src` 差し替え fallback が入っている
