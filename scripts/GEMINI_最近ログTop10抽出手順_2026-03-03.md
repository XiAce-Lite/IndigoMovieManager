# GEMINI向け: 最近ログから処理時間が長い動画Top10を出す手順

## 目的
- `debug-runtime.log` の `task-start/task-end` を使って実処理時間(ms)を算出し、遅い動画Top10を出す。
- 既存スクリプト `scripts/export_thumbnail_log_summary.ps1` を使う（再実装しない）。

## 前提
- 実行場所はリポジトリルート。
- PowerShell 7.x を使う。
- ログは既定で以下を参照する。
  - `%LOCALAPPDATA%\IndigoMovieManager_fork\logs\thumbnail-create-process.csv`
  - `%LOCALAPPDATA%\IndigoMovieManager_fork\logs\debug-runtime.log`

## 実行コマンド（そのまま使ってOK）
```powershell
# 直近6時間を「最近」として集計する例
$from = (Get-Date).AddHours(-6)

pwsh -NoLogo -File .\scripts\export_thumbnail_log_summary.ps1 `
  -TargetDate $from.Date `
  -StartTime $from.ToString("HH:mm:ss") `
  -TopLongCount 10 `
  -AppendTimestamp
```

## 出力の見方
- 出力先は既定で `.\.local`。
- 生成される `thumbnail-log-summary_*.md` の `## 長時間処理動画 best10` が目的の表。
- 表の `DurationMs` が大きい順に並んでいる。

## GEMINIへの指示テンプレート
```text
scripts/export_thumbnail_log_summary.ps1 を使って、直近6時間のログから処理時間が長い動画Top10を抽出してください。
実行コマンドは以下の方針で:
1) from = 現在時刻-6時間
2) -TargetDate from.Date
3) -StartTime from(HH:mm:ss)
4) -TopLongCount 10
5) -AppendTimestamp
出力された markdown から「長時間処理動画 best10」表だけを抜き出して報告してください。
```

## よくある失敗
- `thumbnail-create-process.csv が見つかりません`
  - まだサムネイル作成ログが出ていない。アプリで処理を実行後に再実行する。
- `debug-runtime.log が見つかりません`
  - ランタイムログが未生成。ログ出力設定と実行有無を確認する。
