Ｏ はじめに

一般的なDLLの使い方と一緒です。
必要な構造体や定数等はSinku.hに揃えてありますのでincludeして下さい。

関数のアドレスの取得やバージョン取得はSinkuFunc構造体を使えば楽だと思います。

DLL側ではFILE_INFOの初期化は行っていません。宣言するプログラム側で必ず初期化して下さい。

詳しくは各種言語用のサンプルコードを見て下さい。
今となってはサンプルもかなり古いですが、やろうとしてる事くらいの参考にはなると思います。


基本的にやり取りは全てFILE_INFOを使います。
簡単な使い方の流れは以下の通りです。

1 : DLLをLoadLibraryなりで読み込む
2 : SetPathでCODECS.INIとFORMAT.INIの場所を指定する
3 : FILE_INFOのnameに詳細を取得したいファイル名を入れる
4 : GetFileInfoAutoを呼び出す
5 : ConvertFull、ConvertOne、もしくは自力で、FILE_INFOの中の詳細を表示/使用する
    Convert系を使用する場合はConvert*Lenで予めサイズの取得が可能

※CODECS.INIとFORMAT.INIは両方とも同じ場所に存在する必要があります。


Ｏ Unidode対応について

1005xxのDLLから、Unicode対応となっています。(TCHARを使って切り替えてます)
DLLがUnicode版かANSI版かの判断に Unicode という関数を用意してありますのでそちらで判断してください。

なお、詳細自体は敢えてUnicode化する必要が薄いのでそのまま従来通りShiftJISのままです。
CODECS.INIとFORMAT.INIの内容も同じくShiftJISのままです。


---------------------------------------------------------------------------------------------------
int		GetDllVersion( void )
---------------------------------------------------------------------------------------------------

DLLのバージョンを数値で取得します。
050101なら、50101が返ってきます。

sprintf等でゼロ付6桁揃えで使えばバージョン表示等に使えると思います。
数値の付け方上、数値が下がる事は無いので数値としてバージョン判別にも使えます。


---------------------------------------------------------------------------------------------------
void	SetPath( TCHAR *path )
---------------------------------------------------------------------------------------------------

CODECS.INIとFORMAT.INIのパスを指定します。アプリやDLLと別の場所に*.INIを置く場合は、
初回の「GetFileInfo」か「GetFileInfoAuto」前に必ず設定して下さい。途中での変更はできません。

指定しない場合はアプリと同じ場所の*.INIを参照します。


---------------------------------------------------------------------------------------------------
void	GetFileFormat( FILE_INFO *pFILE_INFO )
---------------------------------------------------------------------------------------------------

ファイルの種別を取得します。
予めFILE_INFOのnameにファイル名を入れておいて下さい。
typeにファイルの種別が入って返ってきます。

種別はSinku.hのenumを見て下さい。


---------------------------------------------------------------------------------------------------
void	GetFileInfo( FILE_INFO *pFILE_INFO )
---------------------------------------------------------------------------------------------------

詳細を取得します。
予めtypeとnameを適切に設定しておいて下さい。
(自前で判別等するのであればGetFileFormatを使わない手もあります)


---------------------------------------------------------------------------------------------------
void	GetFileInfoAuto( FILE_INFO *pFILE_INFO )
---------------------------------------------------------------------------------------------------

GetFileFormatとGetFileInfoを一発で済ませます。
nameさえ指定しておけば詳細が取得されます。

---------------------------------------------------------------------------------------------------
void	ConvertFull( FILE_INFO *pFILE_INFO, char *str )
---------------------------------------------------------------------------------------------------

詳細を真空波動研と同じ形の文字列に組み立てます。
必要なstrの長さはConvertFullLenで取得ができます。


---------------------------------------------------------------------------------------------------
void	ConvertOne( FILE_INFO *pFILE_INFO, char *str )
---------------------------------------------------------------------------------------------------

詳細を真空波動研のプレイリストと同じ形の文字列(1行)に組み立てます。
必要なstrの長さはConvertOneLenで取得ができます。


---------------------------------------------------------------------------------------------------
int		ConvertFullLen( FILE_INFO *pFILE_INFO )
---------------------------------------------------------------------------------------------------

ConvertFullを実行する場合に必要なstrの長さを返します。
末尾のNullはサイズに含まれません。


---------------------------------------------------------------------------------------------------
int		ConvertOneLen( FILE_INFO *pFILE_INFO )
---------------------------------------------------------------------------------------------------

ConvertOneを実行する場合に必要なstrの長さを返します。
末尾のNullはサイズに含まれません。


---------------------------------------------------------------------------------------------------
int		Unicode( void )
---------------------------------------------------------------------------------------------------

DLLがUnicode版であれば1が、ANSI版であれば0を返します。
DLLからこの関数のアドレス自体を取得できなかった場合もANSI版と考えて良いと思います。
